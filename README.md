# AI Website Copy Prompt System

This project demonstrates how **Prompt Engineering + AI tools** can generate high-quality website copy for local businesses.

Many local businesses struggle with:
- unclear website messaging
- weak value propositions
- generic content
- confusing calls-to-action

This prompt system helps generate **conversion-focused website copy** that businesses can directly publish on their websites.

---

## Example Business Used

Business Name: Radiance Salon  
Location: Narsipatnam  
Business Type: Beauty Salon

Target Customers:
- College Students
- Working Professionals
- Families

Services:
- Haircut
- Hair Coloring
- Bridal Makeup
- Hair Spa

---

## Tools Used

AI Tools:
- ChatGPT
- Claude
- Google Gemini

AI Website Builders:
- Lovable AI
- Framer AI

---

## What This Prompt System Generates

1. Homepage website copy
2. Services page content
3. Conversion-focused CTAs
4. Tone adaptation for different businesses
5. Website-ready content

---

## Goal of the Project

To demonstrate how AI can help local businesses improve:

- clarity
- trust
- conversions
- website messaging

This system can be reused for many types of businesses such as salons, cafes, clinics, coaching centers, and agencies.

---

## Repository Files

master_prompt.md – Main prompt that generates full website copy  
homepage_prompt.md – Prompt for homepage content  
services_prompt.md – Prompt for service descriptions  
cta_prompt.md – Prompt for call-to-action sections  
tone_adaptation_prompt.md – Prompt to adjust tone for industries  
website_builder_prompt.md – Prompt to generate website layout  
generated_output.md – Example output for Radiance Salon
