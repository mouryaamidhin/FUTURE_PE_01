# AI Website Copy Prompt System

## Overview
This project uses Prompt Engineering to generate high-quality website copy for local businesses.

---

## 🚀 How to Use

1. Open any prompt file  
2. Paste into ChatGPT / Claude / Gemini  
3. Replace business details  
4. Generate website content  
5. Use in website builders
